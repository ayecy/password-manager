# password-manager
SecureVault - менеджер паролей с архитектурой Zero-knowledge

Ключевые особенности:
-------------------------
Авторизация: регистрация и вход по логину/паролю с BCrypth-хэшированием
-------------------------
Zero-knowledge: мастер-ключ никогда не сохраняется на диск. Работает только в RAM во время сессии пользователя
-------------------------
Криптография: AES-256-GCM + PBKDF2 (100 000 итераций) + уникальная соль на каждого пользователя
-------------------------
CRUD: реализованы в полном объёме в соответствии с криптографией
-------------------------
Поиск: реализован поиск по названию сервиса и/или логина пользователя в реальном времени
-------------------------
Буфер обмена: копирование логина/пароля в один клик с визуальным подтверждением
-------------------------
Генератор паролей: при создании записи логина и пароля для сервиса пользователю предлагается сгенерировать надёжный пароль
-------------------------
Индикатор надёжности: при создании записи логина и пароля для сервиса пользователь видит индикатор "силы" пароля, который зависит от вводимы символов и регистра
-------------------------
Безопасные сессии: JWT с TTL 15 минут, при неактивности автоматическая очистка ключей
-------------------------
Современный UI: Vue 3 + TypeScript, Glassmorphism, анимация частиц, адаптивная верстка на Tailwind


====================================================

Технологический стек:

Backend: Java 21, Spring Boot 3.3.0, Maven
-------------------------
Криптография: BouncyCastle bcprov-jdk18on (PBKDF2, AES/GCM), spring-security-crypto (BCrypt)
-------------------------
Аутентификация: JWT (jjwt-api/impl/jackson 0.12.5) кастомная
-------------------------
Хранение данных: JSON-репозиторий с готовой миграцией на PostgreSQL
-------------------------
Валидация: Bean Validation (@Valid, @NotBlank), DTO-records, строгая проверка на каждом слое
-------------------------
Frontend: Vue 3 (Composition API, <script setup>), TypeScript, Vite 5
-------------------------
Стили и UI: Tailwind CSS, Glassmorphism, SVG-иконки, CSS анимации
-------------------------
HTTP-клиент: Axios 1.10+ с интерцепторами и глобальной обработкой 401
